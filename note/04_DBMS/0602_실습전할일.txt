1. Oracle setup 후, cmd 창을 띄워 Oracle 접속
2. system계정 접속 후, scott 계정 생성 및 권한 부여
C:\Windows\system32>sqlplus system/oracle(설정한 비밀번호)
		          sqlplus "/as sysdba
SQL>create user scott identified by tiger; ( scott/tiger 의 user 생성)

SQL>grant dba to scott;(scott에게 권한 부여 - 최고 권한)
SQL>exit;

3. scott 계정 접속 후, 데이터 확인
C:\~>sqlplus scott/tiger
SQL>SELECT * FROM TAB; - 소유 테이블 확인
