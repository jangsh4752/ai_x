1. 기존의 프로그램 방식
	오차함수 : MSE(오차제곱평균), RMSE(루트를 취하기 때문에 MSE의 단점이 어느정도 해소. 이상치에 덜 민감), MAE(mean absolute erroerror ; 절대값평균)

2. 머신러닝 프로그램 방식
	1. 데이터 확보 및 생성
	2. 데이터 전처리 : 
   		스케일 조정
   	 	훈련데이터(학습데이터셋), 검증데이터셋, 시험데이터셋으로 분할
	3. 모델구성
	4. 모델 학습과정 설정
	5. 모델 학습
	6. 모델 평가(시험데이터셋으로)
	7. 모델 사용(입력값 주어지면 예측값 받기)
